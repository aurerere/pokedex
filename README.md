## Quick start guide
1) Push the SQL script (`@/pokedex.sql`) into a **MySQL** Database
2) Create a `.env` file in `@/api` with those attributes:
    ```py
    HOST = #DB host
    USER_DB = #DB username
    PASSWORD_DB = #DB password
    DATABASE = #DB name
    PORT = #DB port
    ```
3) Go to `@/api` and run:
    ```shell
    $ npm i
    ```
    ```shell
    $ npm run start
    ```

4) Go to `@/front` and run:
    ```shell
    $ npm i
    ```
    ```shell
    $ npm run serve
    ```
